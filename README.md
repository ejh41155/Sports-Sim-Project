# Sports-Sim-Project

This project is a sports player managenment system written in C++. It utilizes a custom player class, where differet players have values such as position, age, and overall rating. 
The players when created are stored in a hashmap for later use. The hashmap has different features such as adding, searching for and displaying player class objects. 
There is a compare player feature where two players can be compared by there different values.
